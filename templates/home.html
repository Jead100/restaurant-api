Restaurant API

Django REST Framework backend for menu, cart, and order management with role-based access.

Quick Links

  • Swagger UI
  • ReDoc
  • GitHub Repository

How to Test Endpoints

  1. Open Swagger UI.
  2. Create or obtain a user token (JWT) via the auth endpoints.
  3. Click "Authorize" in Swagger and paste the token as Bearer <token>.
  4. Exercise menu, cart, and order endpoints according to your role (customer / manager / delivery crew).

Response Shape

All endpoints return a consistent wrapper:

{
  "detail": "Human-readable summary",
  "data": {...} | [...],
  "count": 123,       // only on list endpoints
  "next": "...",     // only on paginated lists
  "previous": null    // only on paginated lists
}

See Swagger for exact schemas.

Auth

JWT-based authentication (SimpleJWT). Rate limits applied per user/IP.
